# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-05-26

### Fixed

- **macOS arm64 crash on launch**: Removed `npmRebuild: false` from `electron-builder.yml` so native addons (`better-sqlite3`, `onnxruntime-node`) are correctly recompiled for the arm64 Electron ABI at package time. Previously the app silently failed because x86_64 Intel binaries were bundled into the arm64 app.
- **Silent crash on DB failure**: `initDb()` no longer throws — failures are logged and the app continues with history features gracefully disabled, preventing a blank-window crash if the native SQLite module cannot load.

## [1.0.0] - 2026-05-22


### Added

- **AI-Powered Presentation Generation**: Users can input prompts to automatically stream slide HTML generated by the Anthropic Claude API, rendered live using `reveal.js`.
- **Offline Local CLI Integration**: Support for offline execution using local system CLI tools (e.g., Gemini CLI) configured with custom model names, system temperature sliders, max tokens, environment variables, and model parameter routing.
- **PowerPoint & HTML Export**: Seamless exporting of generated slide decks to native PowerPoint (`.pptx`) using `pptxgenjs` or fully interactive standalone HTML presentations.
- **Dynamic Slide Editing & Regeneration**: Real-time editor drawer to adjust titles, bullets, and speaker notes per-slide, alongside AI-driven single-slide isolation regeneration with automatic undo capabilities.
- **Local Persistence & Settings Management**: Integrated SQL database (`better-sqlite3`) to save presentation history and `electron-store` for user preferences and API credentials.
- **Silent Background Auto-Updater**: Background update checking and downloading via `electron-updater` with a premium, non-intrusive, dismissible Update Banner notification.
- **Modern Dark-Mode UI**: Sleek styling with custom typography, vibrant violet-to-fuchsia accent gradients, micro-interactions, responsive theme swatches, and global keyboard shortcuts.
